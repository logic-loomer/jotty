# Jotty

Open-source macOS quick-capture app. v0.1 (Phase 1: capture-to-disk foundation).

## Build

Open `Jotty.xcodeproj` in Xcode and run, or:

```bash
xcodebuild -scheme Jotty -destination 'platform=macOS' build
```

## Status

Phase 1 in progress. See `docs/` for the design spec and implementation plan.
